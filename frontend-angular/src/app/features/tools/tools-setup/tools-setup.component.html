Tools Setup Guide

Learn how to download, install, and verify security tools for use in HackSphere

Overview

Before you can run any security tool in HackSphere, you must download and install the tool's executable file. Each tool requires its own binary, script, or executable file to be present on your system.

Important: Tools will not execute unless you have verified their installation. This ensures security and prevents errors from missing dependencies.

Step 1: Browse Tools

All available security tools are pre-configured on the HackSphere server. You do not need to install them locally.

Simplified Access: Simply navigate to the Tools page and click "Run Tool" to start using any available utility.

Step 2: Request New Tools

If a tool you need is missing, please contact the administrator to have it added to the `tools-containerized` directory on the server.

Troubleshooting

Tool not found error

Make sure the file path is correct and the file exists. Check for typos in the path and ensure you're using the correct file extension (.exe, .bat, .py, etc.).

Permission denied

Ensure the tool executable has execute permissions. On Windows, this is usually automatic. On Linux/Mac, you may need to run chmod +x toolname.

Tool requires dependencies

Some tools require additional runtime environments (Python, Java, etc.). Install the required dependencies before verifying the tool.
